Content Syndication

The content syndication passes along the changes to the Main web. You get notified about new and changed pages within CCCSWiki.

Via e-mail

You can receive the recent changes by ElectronicMail?. Add yourself to the list.

Via RSS

Alternatively, the recent changes are also syndicated in RSS 1.0? format. You might want to subscribe to it using an RssReader?.

This feed is available via http and https but the contained links use https. We already apologize that we probably will change this URL once more once we switch off http access.

  • http://ccc-s.bawue.de/wiki/pub/rss/Main/WebChanges.rss

Via Jabber

You can recieve updates via Jabber messanges. Just follow the instructions on JabberServer.
